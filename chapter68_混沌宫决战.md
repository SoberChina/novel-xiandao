# 第六十八章：混沌宫决战

混沌宫的核心区域，是一片浩瀚的虚空。

这片虚空与混沌神殿外的混沌海洋相连，四周是无尽的黑暗与混沌之气。在虚空的正中央，矗立着一座巨大的祭坛。祭坛通体漆黑，表面刻满了密密麻麻的符文，散发着幽幽的光芒。

而在祭坛之上，一道血色的身影正静静地站在那里。

那是血月魔尊。

他周身环绕着浓郁的血光，双眼紧闭，双手结着某种古老的印诀。一道道血色丝线从他体内涌出，连接着祭坛上的符文，形成了一个诡异的阵法。

而在祭坛的正下方，一道巨大的裂缝正在缓缓撕裂。

那裂缝之中，是无尽的黑暗。

那黑暗之中，是混沌魔神沉睡的本体！

陈墨和林清雪赶到的时候，正好看见了这触目惊心的一幕。

"不好！他要打开封印了！"林清雪惊呼。

"住手！"

陈墨低吼一声，身形如同闪电一般朝血月魔尊冲去！

然而就在他即将冲到祭坛前的那一刻，一道血色的屏障忽然从祭坛上升起，将他硬生生地弹了回来！

"哦？来了吗？"

血月魔尊缓缓睁开眼睛，嘴角浮现一丝残忍的笑意。

"混沌圣体的传承者，我等你很久了。"

他转过身来，看着陈墨，眼眸中闪烁着疯狂的光芒：

"你以为你能阻止我吗？太天真了！"

"我已经完成了与混沌本体的连接。再过一刻钟，封印便会彻底崩溃！届时，混沌魔神将会苏醒，整个六界都将被混沌吞噬！"

"而我——将成为混沌魔神在人间的代言人，主宰这个世界！"

陈墨的脸色铁青。

"你做梦。"他的声音冰冷，"我不会让你得逞的。"

"就凭你？"血月魔尊冷笑，"你以为继承了一点混沌神殿的力量，就能与我抗衡了吗？"

"你可知道，我为了这一天，已经准备了数万年！"

他的身上忽然爆发出一道恐怖的血光，那血光之中蕴含的力量，竟然已经超越了天道境！

"这便是混沌魔神赐予我的力量！"血月魔尊狂笑，"在混沌之力面前，你什么都不是！"

恐怖的威压从他身上倾泻而下，压得整片虚空都在颤抖。

林清雪的脸色苍白，身形不由自主地后退了数步。

但陈墨却纹丝不动。

他静静地看着血月魔尊，眼中没有恐惧，只有冰冷的杀意。

"血月，"他缓缓开口，"你知道我先祖陈玄，为什么要在万年前封印混沌魔神吗？"

血月魔尊的笑容微微收敛。

"因为他知道，"陈墨继续说道，"混沌魔神不是什么力量，而是一场灾难。"

"它存在的本身，就是对宇宙秩序的威胁。它想要的不是统治这个世界，而是将一切归于虚无。"

"你以为你是在与它合作？"陈墨冷笑，"你不过是它的一颗棋子。等它苏醒之后，你以为你还能活着？"

血月魔尊的脸色微微一变，但很快便恢复了狂傲：

"那又如何？只要能毁灭这个世界，我愿意付出任何代价！"

"疯子。"陈墨冷冷地说。

"哈哈哈！"血月魔尊大笑，"我是疯子？那这个世界又是什么？"

"我出生在魔界最底层，受尽欺凌与屈辱。我用了数万年时间，才爬到了今天的位置。"

"这个世界对我不仁，我为什么要对它仁义？"

"我就是要看着它毁灭！看着那些曾经看不起我的人，在混沌之中哀嚎！"

他的笑声越来越疯狂，越来越癫狂。

但就在这时，一道冰冷的声音忽然响起：

"你说完了吗？"

血月魔尊的笑声戛然而止。

他抬头望去，只见陈墨已经一步步朝他走来。

每走一步，陈墨身上的气息便强大一分。

当走到血月魔尊面前时，他的周身已经环绕着璀璨的金色光芒——那是混沌圣体的本源之力！

"混沌神殿的力量，不是用来毁灭的。"陈墨的声音平静而坚定，"是用来守护的。"

"先祖用这力量守护了六界万年。"

"现在，轮到我了。"

他的右手缓缓抬起，掌心之中，一道璀璨的金光开始凝聚。

"混沌破天——最终式！"

那道金光携带着毁灭一切的力量，朝血月魔尊轰然砸下！

（第六十八章 完）
