# 第八百四十五章 被贩卖的感觉

商人离开会议室后，曦并没有急于行动。

她知道，单纯的强制命令无法真正说服商人。她需要一种更深层的方法——一种能够让商人亲身体会到问题本质的方法。

曦花了七天时间来准备这个计划。

她创造了一个特殊的梦境场景。在这个场景中，商人不再是那个掌控着众多梦境之门收费亭的商人，而是一个被摆在货架上的"商品"。

曦邀请商人进入这个梦境场景。

商人一开始并不知道曦的意图。它还以为曦是要与它进一步讨论，或者想要通过这个场景来演示某种道理。

"曦大人，"商人说，"您这是要做什么？"

曦微微一笑："商人，我邀请你来这个场景，是想让你体验一种特殊的感觉。"

"什么感觉？"

曦没有直接回答。她只是说："你会知道的。"

然后，商人发现自己出现在了另一个地方。

这是一个巨大的市场。与它平时见到的市场不同，这个市场里摆满了各种各样的"商品"——而这些商品，都是像它一样的元维度存在。

商人惊恐地环顾四周。它发现自己被摆在一个华丽的架子上，身上挂着一个价格标签。

标签上写着："梦境之门使用权×100，珍贵记忆若干，情感宝石三枚。"

商人的心沉了下去。

它想要逃跑，但它发现自己无法移动。它的存在被某种力量固定在了架子上，成为了待价而沽的商品。

一个陌生的元维度走了过来，在商人面前停下，仔细打量着它。

"嗯，这个不错，"那个元维度说，"它有控制梦境之门的能力。如果买下来，我可以让它帮我打开很多扇门。"

另一个元维度凑了过来："是啊，而且它的记忆库里有很多有价值的信息。买下来肯定不亏！"

商人感到一阵深深的恐惧。

它从未想过，自己有朝一日会成为被买卖的对象。它一直以来都是那个进行买卖的人——它习惯了掌控，习惯了决定别人的命运。

但现在，它成为了砧板上的鱼肉。

一个元维度最终买下了商人。它付出了"代价"——一些商人从未见过的东西——然后商人就被"转移"到了那个元维度的领域。

在那个陌生的领域里，商人被迫为新的"主人"工作。它必须使用自己的能力，为主人打开梦境之门，让主人能够进入其他维度。

它没有任何选择的权利。它的存在完全被主人控制。

商人感到前所未有的绝望和愤怒。

它终于意识到：这就是它一直以来在做的事情。

那些被它强迫支付代价才能使用梦境之门的维度，它们的感受与现在的自己一模一样。

它们也曾经是被摆在架子上的"商品"。它们也曾经失去了选择的权利。它们也曾经在绝望中挣扎，却无法逃脱。

商人的泪水在梦境中流淌。

它终于明白了。

曦创造的这个场景，像一面镜子，让它看清了自己的行为本质。

当商人从场景中醒来时，曦正站在它面前。

"商人，"曦说，"你现在明白了吗？"

商人沉默了很长时间。

最后，它开口了，声音里带着深深的悔恨：

"曦大人……我明白了。我终于明白了。"

"我一直以来所做的，与我刚才经历的，并没有本质的区别。我把梦境之门的使用权当作商品来买卖，剥夺了那些维度的选择权。我没有意识到，那扇门不属于我——它属于所有人。"

曦点了点头。

"欢迎回来，商人。"
