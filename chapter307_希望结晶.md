# 第307章 希望结晶

曦月闭上眼睛的那一刻，整个法则之海都安静了下来。

那些漂浮的符文停止了闪烁，仿佛时间本身都被冻结了。

曦月的心跳声清晰可闻，像鼓点一样在寂静中回荡。

"新生之地的力量……"她低声呢喃，"请指引我……回到一万年前的那一天……"

她的身上散发出越来越强烈的光芒，那光芒温暖而柔和，像是春天的阳光，又像是母亲的呢喃。

曦紧紧握着曦光的手，指节因为用力而发白。

她想冲上去阻止曦月，但她知道已经来不及了。

曦月已经将自己的意识，完全沉浸在了因果的长河之中。

一万年前。

法则之海边缘的一个小维度。

因果魔君——那时候还叫做因果守护者——正站在他的小屋前，看着妻子和三个孩子在院子里嬉戏。

阳光洒在他们身上，将一切都镀上了一层金色。

因果守护者的脸上，洋溢着幸福的笑容。

那时候的他，还不知道灾难即将降临。

就在这个时候，一道黑影突然出现在了法则之海的方向。

因果守护者的表情瞬间变得严肃起来。

"黑暗入侵！"他低喝一声，"我必须去阻止！"

他转身就要离开，但他的妻子却拉住了他的手。

"你一定要回来。"她温柔地说。

因果守护者看着她的眼睛，微微笑了。

"我一定会回来。"

然后，他头也不回地飞向了法则之海。

那是他最后一次看到妻子的笑容。

——

与此同时，在法则之海中，曦月正在穿越时间的洪流。

她能感觉到，因果的长河在她的身边流淌，每一滴水珠都是一个因果，一个故事。

她逆流而上，穿过无数个因果节点，终于来到了一万年前的那个时刻。

她看到了因果守护者，看到了那场即将发生的战斗。

她也看到了那道因果之力——那道即将击中他家人的因果之力。

曦月深吸了一口气，然后将自己的力量注入了那道因果之力之中。

"改变……"她低声说道，"我要改变这道因果……"

她的力量与因果之力交融在一起，像是两条河流汇聚成了汪洋大海。

那原本会击中因果守护者家人的因果之力，在曦月的作用下，改变了方向。

它击中了一块空旷的空地，激起了一阵烟尘，然后消散在了空气之中。

而因果守护者的家人，平安无事。

曦月露出了一个欣慰的笑容。

成功了。

但就在这个时候，一股强大的反噬之力突然袭来。

那是因果的反噬。

曦月改变了因果，她就必须为此付出代价。

那股力量狠狠地撞入了曦月的身体，像是要将她整个人都撕裂。

曦月痛苦地皱起了眉头，但她没有退缩。

"这是……我自己的选择……"她咬紧牙关，"我无怨无悔……"

她的身体在那股反噬之力中变得越来越虚弱，意识也开始变得模糊。

但她始终没有放弃。

她要亲眼看到因果魔君的家人平安无事，她才愿意离开。

时间继续流逝。

曦月看到因果守护者击退了黑暗力量，回到了家中。

他看到了他的妻子和三个孩子，平安无事地站在院子里等他。

他的脸上露出了惊喜和感激的泪水。

"你们……你们没事？"他的声音颤抖。

"我们没事。"他的妻子微笑着说，"你回来了。"

曦月看着这一幕，终于露出了一个安心的笑容。

她的使命……完成了。

然后，她闭上了眼睛。

她的意识缓缓脱离了时间的洪流，回到了现实之中。

——

当曦月再次睁开眼睛的时候，她发现自己躺在了曦的怀里。

曦的眼眶通红，显然刚刚哭过。

曦光和念石站在一旁，神情同样沉重。

"曦月！"曦看到她醒来，惊喜地叫了出来，"你终于醒了！"

曦月微微笑了笑，想要坐起来，但身体却传来一阵剧痛。

她低头看去，只见自己的胸口处，有一道黑色的纹路正在缓缓蔓延。

那是因果反噬的痕迹。

"曦月！"曦看到了那道黑色纹路，脸色瞬间变得苍白，"这是……"

曦月轻轻摇了摇头。

"没关系。"她的声音很轻，"这是我自己的选择。"

而在不远处，因果魔君呆呆地站在那里，眼睁睁地看着这一切。

他看到了曦月为了改变他的过去，承受了多大的代价。

他的心中，涌起了一股前所未有的感动。

——

**第三百零七章 完**
