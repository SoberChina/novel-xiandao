# 第一百二十九章：内部叛徒

林婉儿在轮回深渊边缘，与幽冥守护者对峙。

修复轮回系统——这绝非易事。

但为了拯救三界，她没有退路。

"轮回系统的核心，是轮回池。"幽冥守护者缓缓说道，"当年混沌宫覆灭时，轮回池被一股邪恶的力量侵蚀，导致轮回法则紊乱。"

"这三万年来，无数魂魄无法正常轮回，要么滞留在凡间化为孤魂野鬼，要么直接消散在天地之间。"

"幽冥界也因此日益衰落。"

林婉儿沉默了。

她能感受到幽冥界弥漫的那股衰败之气——这里曾经是万物归宿之地，如今却成了被遗忘的角落。

"我该怎么做？"她问道。

"进入轮回池，从内部修复被侵蚀的法则。"幽冥守护者说道，"但轮回池中充斥着混乱的轮回之力，稍有不慎就会被卷入其中，万劫不复。"

"你有这个勇气吗？"

林婉儿没有犹豫。

"为了三界，为了小石头，为了所有在混乱中受苦的生灵——"

"我愿意付出一切。"

幽冥守护者看着她，眼中闪过一丝动容。

"好。"

他抬起手，一道黑色的门户在深渊边缘浮现。

"轮回池就在里面。"

"去吧。"

林婉儿深吸一口气，正要踏入门户——

突然，一道急促的声音从远处传来。

"林姑娘！等等！"

她转头望去，只见一道狼狈的身影正向她飞来。

是太虚！

但此刻的太虚浑身浴血，气息紊乱，显然受了重伤。

"太虚！"林婉儿大惊，连忙上前扶住他，"你怎么伤成这样了？！"

"那些黑袍人……"太虚喘着粗气，"他们的实力……远超我的想象……"

"而且他们使用的混沌之力……与虚无之力融合之后……变得异常恐怖……"

"我拼尽全力才逃出来……但其他人……"

他的声音越来越弱，显然已经撑不住了。

林婉儿连忙将一股四力之源的力量渡入他体内，稳住他的伤势。

"你先休息。"她说道，"剩下的事情交给我。"

太虚点了点头，但突然又挣扎着睁开眼睛。

"林姑娘，还有一件事……"

"我逃出来的时候……发现了那些黑袍人的身份……"

林婉儿心中一凛。

"什么身份？"

"他们……不是外面的人……"太虚的声音越来越弱，"他们……是天道殿的……"

话未说完，太虚便昏了过去。

林婉儿如同被雷击中一般，僵在原地。

天道殿的人？

那些使用混沌之力与虚无之力的黑袍人，竟然是天道殿的人？

这怎么可能？！

天道殿是小石头一手建立起来的，是为了守护天地秩序而存在的。

怎么可能会有叛徒？

"看来你已经意识到了。"幽冥守护者突然开口。

林婉儿猛然抬头。

"您知道什么？"

幽冥守护者沉默了片刻。

"当年混沌宫的覆灭，并非天帝一人所为。"

这句话如同一道惊雷，在林婉儿脑海中炸响。

"您说什么？"

"我说，"幽冥守护者的声音变得低沉，"当年混沌宫的覆灭，是天帝与某个势力联手所致。"

"那个势力……就隐藏在你们天道殿之中。"

"而且——"他的目光变得锐利，"直到今天，那个势力依然存在。"

林婉儿只觉得浑身发冷。

天道殿内部，有混沌宫的叛徒？

而且这些叛徒，还在暗中破坏她寻找法器的行动？

"这就是为什么，那些黑袍人会使用混沌之力与虚无之力。"幽冥守护者说道，"因为他们本来就是混沌宫的余孽。"

"混沌宫覆灭时，有一部分人逃了出来，隐藏在暗处。"

"他们一直伺机而动，等待着复仇的机会。"

"而现在，随着小石头的离去，天道殿力量空虚——"

"他们的机会来了。"

林婉儿紧握双拳。

"告诉我，是谁？"

"我不知道具体是谁。"幽冥守护者摇头，"但我知道一件事——"

"那个势力在天道殿内部经营了数万年，根深蒂固。"

"你信任的人中，很可能就有他们的人。"

"你必须小心。"

林婉儿沉默了。

她想起了进入天道殿以来的种种——那些并肩作战的战友，那些共同经历生死的伙伴……

其中竟然有人是叛徒？

她不敢相信，但太虚的话不会有错。

那些黑袍人确实使用的是混沌之力与虚无之力——这绝对不是外人能够轻易获得的力量。

"我明白了。"她深吸一口气，"谢谢您告诉我这些。"

"我会小心的。"

她低头看向昏迷中的太虚。

"太虚，你先在这里养伤。"

"我要回一趟天道殿。"

"必须查清楚，叛徒究竟是谁。"

她站起身，目光坚定。

"还有造化炉——"她看向幽冥守护者，"等我回来。"

"我会证明我有资格拥有它。"

幽冥守护者看着她，微微点头。

"我等你。"

---

三日后。

天道殿，议事大殿。

林婉儿回到天道殿后，立刻召集了所有核心成员。

陈墨、林清雪、鸿钧、妖族族长、魔尊……

所有能联系到的人，都来到了大殿。

林婉儿站在大殿中央，目光扫过每一个人。

"诸位，"她开口，声音沉稳，"我有一个重要的事情要宣布。"

"三界融合的根源，是因为维持三界平衡的五大法器散落各地。"

"我已经找到了混沌钟和阴阳镜。"

"星辰图也已经在我手中。"

"造化炉在幽冥界，我正在争取。"

"但现在，我们遇到了一个巨大的阻碍。"

她的目光变得锐利。

"天道殿内部，有叛徒。"

此言一出，满堂哗然。

"什么？！"

"这怎么可能？！"

"谁？是谁？！"

众人议论纷纷，神情各异。

林婉儿仔细观察着每一个人的反应。

陈墨神色凝重，似乎在思考着什么。

林清雪满脸震惊，第一反应是不相信。

鸿钧眉头紧锁，一言不发。

妖族族长和魔尊对视一眼，眼中闪过一丝警惕。

每个人的反应都值得玩味，但林婉儿一时也无法判断谁是叛徒。

"这些叛徒使用混沌之力与虚无之力融合的邪气，伏击了我和太虚。"她继续说道，"太虚重伤，目前在幽冥界养伤。"

"他们的目的很明确——阻止我集齐五大法器，阻止我控制三界融合。"

"他们想要让宇宙毁灭。"

"而他们的身份，与当年混沌宫的覆灭有关。"

此话一出，陈墨的脸色微微一变。

"你是说……混沌宫的余孽？"

"没错。"林婉儿点头，"幽冥守护者告诉我，当年混沌宫覆灭，并非只是天帝一人所为。"

"而是天帝与某个隐藏在暗处的势力联手。"

"那个势力……一直隐藏在天道殿之中。"

"直到今天。"

大殿陷入了死一般的寂静。

每个人都在消化这个惊人的消息。

陈墨沉默良久，然后开口：

"林婉儿，你有证据吗？"

"没有直接证据。"林婉儿摇头，"但太虚亲眼看到那些人使用混沌之力与虚无之力。"

"而且那些人在伏击我们时，使用了一种极其诡异的功法——与混沌宫的功法有着惊人的相似。"

"这种力量，不是外人能够拥有的。"

陈墨沉默了。

他的表情变得复杂，似乎在回忆着什么。

良久，他叹了口气。

"其实……我早就有所怀疑了。"

众人看向他。

"这些年，天道殿内部总有一些事情不对劲。"他缓缓说道，"很多决策，在执行过程中会出现意外。"

"我一直以为是天意，但现在想来……"

"很可能是有人从中作梗。"

林婉儿看着他。

"陈墨前辈，您是在什么时候开始怀疑的？"

陈墨沉吟片刻。

"大约在……小石头离开之前不久。"

"有一次，我布置了一个任务，派了一队弟子去调查混沌源地的异常。"

"但那队弟子在半路上遭遇了伏击，全军覆没。"

"当时我以为是混沌源地的危险所致，但现在想来……"

他的声音变得低沉。

"很可能有人走漏了消息。"

众人再次陷入沉默。

如果陈墨所言属实，那说明天道殿内部的叛徒势力，远比他们想象的更加庞大。

而且这个势力，很可能已经渗透到了核心层。

"林婉儿。"陈墨突然开口。

"你怎么突然提起这件事？"

"是因为，"林婉儿看着他，"我收到了一封匿名信。"

她从怀中取出一张纸条。

"就在我离开幽冥界之前，有人将这封信塞到了我房门口。"

"信上写着：天道殿内部，叛徒不止一人。其中一人，地位极高。"

"想要找到叛徒，先查当年的账簿。"

"账簿？"众人疑惑。

"什么账簿？"

林婉儿摇头。

"我也不知道。"

"但我知道一件事——"

"这封信的目的，是引导我找到叛徒的线索。"

"不管写这封信的人是敌是友，我都要追查下去。"

"当年的账簿——"

"究竟记录了什么？"
