# 第五十二章：混沌十凶

踏入门户的瞬间，陈玄只觉天旋地转。

一股巨大的拉扯之力将他卷入其中，四周的一切都在飞速旋转。

待他回过神来，已经身处一片完全不同的世界。

这里的天空是灰蒙蒙的，没有日月星辰。

大地是一片漆黑的平原，寸草不生。

空气中弥漫着浓郁的混沌之力，比混沌领域外围还要浓郁十倍不止。

"这里就是混沌领域的最深处吗..."陈玄喃喃道。

他感受到，这里的混沌之力几乎凝成了实质，如同实质的墙壁一般压在他的身上。

普通人若是身处此地，只怕瞬间便会化为灰烬。

但陈玄体内有混沌本源，对混沌之力有着天然的抗性。

他深吸一口气，适应着这里的环境。

就在这时，远处突然传来一声惊天动地的咆哮！

"吼——！"

那声音中蕴含着恐怖的力量，震得陈玄脚下的平原都在颤抖。

"好强的气息..."陈玄的脸色微变。

他感应到，那股气息的强度，竟然不弱于道境界巅峰的强者！

"难道是..."陈玄心中一动。

下一刻，一道巨大的身影从远处的平原上拔地而起！

那身影高达千丈，通体漆黑，周身萦绕着浓郁的混沌之力。

它的形态似虎非虎，似狮非狮，背后还生着九根巨大的骨刺，散发着幽幽的光芒。

"混沌凶兽——噬天！"陈玄瞳孔骤缩。

他在古籍中看到过关于噬天的记载。

噬天是混沌领域中最强大的凶兽之一，据说拥有毁天灭地的力量。

它的一声咆哮，便能震碎一方小世界。

而眼前这只噬天，气息之强大，远超陈玄的想象。

"不对..."陈玄仔细感应着噬天的气息，"这只是噬天的分身，并非本体..."

噬天乃是混沌凶兽中的佼佼者，实力不弱于道境界巅峰的存在。

它的本体，深藏在混沌深处的巢穴之中，不会轻易出动。

而眼前这只，应该只是噬天的一个分身。

但即便只是一个分身，其实力也足以与道境界后期的强者抗衡！

"吼——！"

噬天再次发出一声咆哮，巨大的身影向陈玄扑来！

陈玄不敢大意，连忙催动体内的混沌之力，与噬天战在一起！

"轰——！"

两人交手的第一击，便产生了毁天灭地的效果。

恐怖的气浪向四面八方席卷，将方圆万里的平原都削去了一层。

"好强..."陈玄的脸色微变。

虽然这只是一具分身，但噬天的力量依然强大得可怕。

每一次交手，都让陈玄感到手臂发麻。

而噬天似乎不知疲倦，越战越勇。

"这样下去不是办法..."陈玄眉头一皱。

他与噬天的实力本就在伯仲之间，若是久战不下，必然会陷入被动。

更何况，这里是混沌领域的最深处，谁也不知道还有多少危险在等着他。

必须速战速决！

陈玄眼中闪过一抹决然，做出了一个决定。

他要施展《混沌真经》中的禁忌之术——混沌归元！

混沌归元，是将体内的混沌之力全部释放出来，产生毁天灭地的攻击。

但这种秘术对身体的负荷极大，若是使用不当，甚至可能伤及本源。

但此刻，陈玄已经顾不了那么多了。

"混沌归元！"

他大喝一声，体内的混沌之力如同洪水一般倾泻而出！

刹那间，天地变色！

一股恐怖到极点的力量从陈玄体内爆发出来，带着毁灭一切的气息，向噬天席卷而去！

"吼——！"

噬天发出一声惊恐的咆哮。

它感受到了那股力量的可怕，想要逃跑，却已经来不及了。

"轰——！"

惊天动地的爆炸声响彻整个平原。

噬天的分身在这股力量面前，如同纸糊一般脆弱，瞬间便化为灰烬。

而陈玄，则是大口大口地喘着粗气，脸色苍白。

混沌归元的负荷太大，即便以他的身体素质，也感到极为疲惫。

"总算解决了..."陈玄喃喃道。

但就在这时，他的脸色突然一变。

因为他感应到，远方又有数道强大的气息正在飞速接近！

不仅如此，那数道气息的强度，竟然都不弱于噬天！

"不好..."陈玄的脸色变得极为难看。

他突然想起了古籍中关于混沌领域最深处的一段记载。

在混沌领域的最深处，有十只混沌凶兽，被称为"混沌十凶"。

它们是混沌魔神亲手创造的凶兽，实力强大到恐怖。

每一只都有不弱于道境界巅峰的实力！

而刚才被陈玄消灭的噬天，正是混沌十凶之一！

"难道说..."陈玄的心猛地一沉。

下一刻，他的猜测便得到了证实。

九道巨大的身影从四面八方出现，将陈玄团团围住。

这九道身影，每一道都散发着不弱于噬天的气息。

它们的形态各异，但都强大得可怕。

"混沌十凶，竟然来了九个..."陈玄的眼中闪过一抹苦笑。

他刚才施展混沌归元，消耗了大量的力量。

现在的他，恐怕连全盛时期的一半实力都发挥不出来。

而眼前却有九只道境界巅峰级别的凶兽！

这一战，该如何打？

"诸位..."陈玄深吸一口气，开口道，"我并无恶意，只是想寻找混沌神殿..."

"吼——！"

回应他的，是九只凶兽齐声咆哮！

它们似乎根本不想听陈玄解释，直接便扑了上来！

"既然如此..."陈玄的眼中闪过一抹寒光，"那就战吧！"

他催动体内剩余的力量，与九只凶兽战在了一起。

但这一次，他不再硬拼。

而是利用灵活的身法，在九只凶兽之间穿梭，寻找着每一个可能的破绽。

《混沌真经》在体内疯狂运转，将周围的混沌之力不断吸入体内，转化为陈玄的力量。

渐渐地，陈玄开始适应了这种战斗的节奏。

他的脑海中，也在飞速思考着对策。

九只凶兽虽强，但它们毕竟不是真正的混沌魔神。

若是能够找到它们的弱点，便有机会破局。

而就在这时，陈玄突然发现了一个关键的问题。

这九只凶兽的站位，似乎遵循着某种规律。

它们的位置，连成了一条条隐形的线，隐隐形成了一个阵法。

"阵法..."陈玄的眼中闪过一抹精光。

若是将这些凶兽看成阵法的节点，那这个阵法的核心在哪里？

他的目光在九只凶兽身上扫过，最终定格在其中一只凶兽身上。

那是一只形态似蛇的凶兽，名叫"噬魂"。

与其他凶兽的狂暴不同，噬魂的气息极为阴冷，仿佛来自九幽地狱。

而更重要的是，它一直躲在其他凶兽身后，从不正面进攻。

"噬魂...你便是这个阵法的核心吗？"陈玄喃喃道。

他不再犹豫，身形一闪，向噬魂冲去！

"吼——！"

其他八只凶兽见状，纷纷怒吼着拦截。

但陈玄早有准备，他催动体内的混沌之力，凝聚出一道强大的护盾，挡住了八只凶兽的攻击。

而他自己，则是趁着这一瞬间的空档，杀到了噬魂面前！

"混沌破天！"

陈玄大喝一声，一拳轰向噬魂！

噬魂发出一声惊恐的尖叫，想要躲避，却已经来不及了。

"砰——！"

陈玄的一拳，正中噬魂的要害！

噬魂发出一声凄厉的惨叫，庞大的身躯开始崩溃！

而随着噬魂的崩溃，那个隐形的阵法也随之瓦解！

失去了阵法的加持，其他八只凶兽的战力瞬间大减。

陈玄趁势追击，施展出《混沌真经》中的终极招式。

"万法归一！"

刹那间，无数道混沌之力从陈玄体内涌出，化为一道毁天灭地的光柱，向八只凶兽席卷而去！

"轰轰轰——！"

连续八声爆炸声响彻天际。

八只凶兽先后陨落，化为漫天的混沌之气，消散在虚空之中。

"呼..."陈玄长出一口气，瘫坐在地上。

这一战，实在太过凶险。

若是稍有不慎，他便会葬身于此。

但好在，他赢了。

不仅如此，在与混沌十凶的战斗中，他的修为再次提升。

他已经触碰到了道境界巅峰的门槛。

只要再进一步，便能突破到道境界巅峰！

"看来，机缘就在危险之中..."陈玄喃喃道，脸上露出一丝苦笑。

他站起身来，继续向混沌深处走去。

而前方，混沌神殿的轮廓，已经隐隐可见。

（本章完）
