# 《仙道永续》

> 混沌圣体传承，万年仙魔之战，仙道永续

---

## 简介

苍茫大陆，亿万生灵繁衍生息。传说天地初开，六界便已成形——人界、魔界、妖界、仙界、佛界、神界。

在人界，修士与凡人共存。修炼之道，共分十三重境界：炼气、筑基、金丹、元婴、化神、渡劫、大乘，真仙、金仙、太乙金仙、大罗金仙、道祖、天道境。

灵根，是决定命运的第一步。灵根分为下品、中品、上品、极品四个层次。拥有极品灵根者，整个苍茫大陆百年都未必能出一个。

而主人公陈墨，正是那个百年难遇的极品灵根拥有者——同时，他体内还封印着远古最强大的体质之一：**混沌圣体**。

这是一个关于守护、传承与信念的故事。

---

## 世界观

- **修炼境界**：十三重（炼气 → 天道境）
- **灵根等级**：下品 / 中品 / 上品 / 极品
- **六界**：人界、魔界、妖界、仙界、佛界、神界

---

## 角色

- **陈墨** — 清溪村少年，混沌圣体传承者，"以记为生"的书呆子修士
- **陈老实** — 陈墨的爷爷，身份成谜的神秘老人
- **林清雪** — 青云宗内门弟子，冷若冰霜却心细如发的天才少女
- **周长风** — 青云宗外门长老，发现陈墨资质的伯乐
