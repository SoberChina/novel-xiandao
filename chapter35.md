# 第三十五章：身份暴露

站在宫殿门口的黑袍老者，正是万魔宗幕后黑手——魔界暗皇！

“你怎么会在这里？！”陈玄脸色大变。

“本座为何不能在这里？”暗皇冷笑，“这片虚无之渊，本就是本座的领地。”

“你的领地？”陈玄瞳孔骤缩，“你与混沌魔神是什么关系？”

“关系？”暗皇笑了，“告诉你也无妨。本座体内，流淌着混沌魔神的一缕血脉！”

“什么？！”

“当年混沌魔神被封印时，曾分裂出数道意念，本座便是其中之一。”暗皇傲然道，“论血脉，本座比你这个所谓混元仙帝之子高贵得多！”

陈玄恍然大悟。

难怪万魔宗如此强大，原来其幕后黑手竟然与混沌魔神有关！

“当年仙魔大战，本座本想趁乱挣脱封印，没想到混元仙帝那个叛徒竟然与仙族联手，将本座重伤！”暗皇怒道。

“可惜，混元仙帝也死了，而本座也陷入了沉睡...”

“直到百年前，本座才逐渐苏醒。本以为万魔宗能帮本座重新崛起，没想到...”暗皇看向陈玄，眼中满是杀意，“没想到混元仙帝的儿子竟然还活着，而且还得倒了道祖的传承...”

“天意啊！真是天意！”暗皇大笑，“今日，本座就要亲手杀了你，用你的血祭奠本座这些年的屈辱！”

话音落下，暗皇一掌拍出！

“小心！”陈玄大喝一声，催动古玉迎战。

“轰——！”

两人的攻击在空中碰撞，爆发出惊天动地的能量风暴！

陈玄被震得连连后退，口中一甜，差点吐血。

暗皇的实力，果然深不可测！

“小子，你以为打败了混沌魔神的分身，就能与本座抗衡？”暗皇冷笑，“那分身不过是本座的一道意念所化，你打败它不算什么！”

“受死吧！”

暗皇再次出手，这一次他不再保留！

无数黑色的光芒从他身上射出，化为一张大网，向陈玄笼罩而去！

“道祖剑法——斩神！”

陈玄拼尽全力，施展出刚才学会的最强剑招！

然而这一次，暗皇早有准备。

只见他双手结印，一道黑色的屏障出现在身前，竟然挡住了陈玄的剑光！

“什么？！”陈玄大惊。

他的剑光竟然被挡住了！

“道祖剑法固然强大，但你修为太弱，发挥不出其真正的威力！”暗皇冷笑，“今日，你必死无疑！”

黑色的光芒大网越来越近，陈玄拼尽全力也无法挣脱。

“怎么办？！”陈玄心中焦急。

就在这千钧一发之际，他体内的古玉突然爆发出一道璀璨的金光！

金光直接冲破了黑色大网的束缚，并向暗皇袭去！

“什么？！”暗皇脸色大变，急忙后退。

金光之中，一道威严的身影浮现——正是道祖！

“暗皇，多年不见，你还是这般不知死活。”道祖的声音冰冷。

“道祖？！”暗皇脸色大变，“你明明已经陨落了，为何还有残念留存？！”

“老夫虽然陨落，但岂会不留后手？”道祖冷笑道，“当年你趁老夫与混沌魔神大战时偷袭，今日，老夫便与你算清这笔账！”

话音落下，道祖的残念突然爆发出惊人的光芒！

“不妙！”暗皇脸色大变，转身就跑。

“想跑？”道祖冷笑，“晚了！”

一道金光射出，直接追上暗皇！

“噗！”暗皇惨叫一声，身躯被金光洞穿！

“可恶...可恶！”暗皇怒吼，“道祖，你等着，总有一天本座会找你算账！”

话音落下，暗皇的身影突然炸裂，化为无数黑色的光点消散。

但陈玄知道，暗皇并没有死，只是暂时逃遁了。

“多谢前辈救命之恩！”陈玄躬身道。

“莫要谢我。”道祖的残念叹道，“老夫这道意念也即将消散，以后就要靠你自己了...”

“前辈！”陈玄急道。

“放心。”道祖道，“老夫已将毕生所学都刻入你的识海，关键时刻，这道意念会自行激发保护你。但下一次，可就不一定有这么好运了...”

“老夫的时间不多了...记住老夫的话——”

“混元仙帝之子和道祖传人，这两个身份是你最大的优势，也是最大的弱点。好好利用你的血脉，它会给你意想不到的力量...”

“还有，暗皇虽然重伤，但他不会善罢甘休。你要尽快提升实力，否则下次见面，你必死无疑...”

话音越来越淡，最终消失。

道祖的残念，彻底消散了。

陈玄站在原地，心中感慨万千。

这一趟虚无之渊之行，虽然险象环生，但收获也是巨大的。

他不仅得到了道祖的完整传承，还知道了自己真正的身世——他是混元仙帝之子！

更重要的是，他知道了自己将来的使命——同化混沌魔神，让混沌归于秩序！

“暗皇...”陈玄握紧拳头，“下次见面，我绝不会输给你！”

他转身离开宫殿，向着来时的方向走去。

他现在需要尽快回到天玄宫，将这一切告诉众人，然后继续闭关修炼。

只有变得更强，才能保护自己想要保护的人，才能完成道祖和父亲的遗愿！

然而他不知道的是，在他离开后不久，那座封印阵法突然剧烈震动起来...

混沌魔神的本体，似乎察觉到了什么...
